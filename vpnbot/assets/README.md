# assets/

Каталог изображений для сообщений бота.

## Как заменить картинку

Просто положи (или замени) файл с нужным именем (`.jpg`, `.jpeg` или `.png`) — бот подхватит изменение
**без перезапуска**: каждое отправляемое сообщение читает файл из этой папки.
Если файла нет — бот пришлёт сообщение без картинки (только текст).

## Имена файлов (один из расширений: `.jpg` / `.jpeg` / `.png`)

| Файл | Где используется |
|---|---|
| `main`           | Главное меню (приветствие после /start) |
| `plans`          | Выбор тарифа |
| `period`         | Выбор периода (длительности) |
| `payment`        | Выбор способа оплаты |
| `payment_link`   | Шаг с ссылкой на оплату (после выбора ЮKassa/CryptoBot) |
| `profile`        | Профиль пользователя |
| `info`           | Раздел «Информация» |
| `referral`       | Реферальная программа |
| `promo`          | Ввод промокода |
| `trial`          | Пробный период (подписка на канал) |
| `admin`          | Админ-панель — главное меню |
| `admin_users`    | Админка → Пользователи |
| `admin_promo`    | Админка → Промокоды |
| `admin_broadcast`| Админка → Рассылка |
| `admin_pricing`  | Админка → Тарифы |

## Рекомендации

- Размер 1280×720 px, оптимально 1600×900.
- Вес — до 1 МБ (Telegram сжимает, но мелкий файл = быстрее загрузка).
- Расширение `.jpg` — приоритет; затем `.jpeg`, `.png`.
